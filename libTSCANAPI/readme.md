如果需要本库能直接集成进入Python-can

只需要将config.py文件中的IS_ADD_PYTHON_CAN设置为True即可

注意：IS_ADD_PYTHON_CAN = True后需要调用本库运行一次，将配置文件写入到python-can中，之后就可以直接使用，不需要再做此配置
